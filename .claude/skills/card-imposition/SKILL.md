# 스킬: 카드 및 명함 양면 조판 (Card Imposition Skill)

이 스킬은 명함재단기(Slitter) 연동을 위한 인쇄용 PDF 조판 로직을 설계하고 구현하는 데 특화되어 있다.

## 1. 핵심 원칙
- **동적 마커 생성**: 센서 마커의 위치와 크기는 항상 관리자 설정에서 가져온다. **절대 하드코딩하지 않는다.**
- **양면 좌우 반전**: 앞면과 뒷면은 가로축 기준 Mirroring이 필수다.
- **왼쪽 에지 기준**: 모든 좌표(마커 및 카드)는 재단 가이드인 왼쪽 에지를 기준으로 산출한다.

## 2. 세부 지침
- 조판 엔진은 실행 시 `config.marker_x`, `config.marker_y` 등의 변수를 주입받아야 한다.
- `use_sensor_marker`가 false인 경우 마커 드로잉 과정을 스킵한다.
- 뒷면의 열(Column) 배치 시 `(Total_Columns - current_col + 1)` 식을 사용하여 위치를 반전시킨다.

## 3. 에이전트 체크리스트
- [ ] 마커 좌표가 설정 변수로부터 동적으로 할당되는가?
- [ ] 뒷면 조판 시 카드 배치가 앞면과 좌우 대칭을 이루는가?
- [ ] 마커 색상이 K100(검정)으로 지정되었는가?
