초안 및 분석 (Draft & Analysis)

초안: 기존 PHP 쇼핑몰에 React 편집기를 임베딩하고, 무거운 PDF 처리는 Node.js 워커가 담당하는 3티어 구조로 시작되었습니다.

분석: 5월 초 진행된 통합 오딧(Audit)을 통해 멀티사이트 대응을 위한 API 인증 강화와 에디터 컴포넌트의 과도한 결합도가 문제점으로 지적되었습니다.

대응 및 개선 (Response & Improvements)

  • 모듈화 완료: 1,500줄의 단일 파일을 18개 파일로 분리하여 관리 효율을 극대화했습니다.
  • UI/UX 개편: 객체 선택 핸들 디자인을 직관적으로 변경하고, 캔버스 자동 중앙 정렬을 도입했습니다.
  • 호환성 확보: snake_case와 camelCase를 모두 지원하는 파라미터 헬퍼를 도입하여 PHP 레거시 연동성을 높였습니다.
