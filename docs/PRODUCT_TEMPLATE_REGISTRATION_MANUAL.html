1에셋(디자인 리소스) 등록 — Library

관리자 메뉴 Library 에서 고객 편집기에 공급할 디자인 리소스를 업로드합니다. 업로드 즉시 고객 편집기의 해당 패널에 노출됩니다 2026-06-03 연결됨.

에셋 종류관리자 화면편집기 노출 패널저장 테이블
클립아트(요소)Library › 클립아트요소library_cliparts
사진틀/프레임Library › 프레임프레임library_frames
배경Library › 배경배경library_backgrounds
도형Library › 도형도형library_shapes
폰트Library › 폰트텍스트(폰트 선택)library_fonts
권장 규격 · 인쇄 화질을 위해 사진틀/배경 PNG 는 300 DPI 기준(예: A4 ≈ 2480×3508px) 권장. 편집기→PDF 합성 시 최대 3508px 까지 원본 화질을 보존합니다.
주의 · 에셋은 카테고리/태그를 지정하면 고객 패널에서 검색·필터가 됩니다. isActive=false 인 에셋은 고객에게 노출되지 않습니다(임시 비공개).

2템플릿셋(TemplateSet) 생성

관리자 메뉴 TemplateSets › 새 템플릿셋. 템플릿셋은 "한 상품을 만들기 위한 판형·페이지·도구 규칙의 묶음"입니다.

필드의미비고
템플릿셋명관리/선택용 이름필수
판형/규격 printSize완성 크기(mm) + 재단(bleed)PDF mediabox 기준
canAddPage / pageCountRange고객 페이지 추가 허용 + 최소/최대PDF 첨부 시 페이지수 자동 맞춤
coverEditable표지 편집 가능 여부false=레더커버/화보집(미리보기 이미지만)

3템플릿(캔버스 디자인) 제작

관리자 메뉴 Templates › 편집기(관리자 모드). 표지/내지 캔버스에 1단계 에셋을 배치해 고객이 시작할 디자인을 만듭니다. 관리자 모드(editMode)로 진입하면 잠금·삭제잠금 등 제작 전용 기능이 활성화됩니다.

배치할 수 있는 요소

  • 레이아웃 / 배경 / 사진틀(프레임)
  • 텍스트(부분 색상·기울임·크기·곡선/원형 원형 배지)
  • 요소(클립아트) / 도형 / 이미지

객체 보호 지정 2026-06-03

  • 이동 잠금: 자물쇠 버튼 또는 cmd+L
  • 삭제 잠금: ControlBar 방패 아이콘(관리자 전용)
  • 저장→복원 후에도 보호 유지 → 고객은 이동·삭제 불가
WYSIWYG 합성 · 고객이 편집을 완료하면 화면 캔버스가 300 DPI 벡터/원본화질로 PDF 합성됩니다. 표지 PDF + 앞면지 + 내지 + 뒷면지가 하나로 합본됩니다.
