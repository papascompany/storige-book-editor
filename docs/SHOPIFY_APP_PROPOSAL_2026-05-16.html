2 Shopify 앱의 3가지 임베드 위치

Storige는 3개 모두 사용합니다. 각각의 역할이 명확히 분리.

REQUIRED

Admin Embedded App

📍 Shopify Admin (판매자)
App Bridge + Polaris UI. 판매자가 사이트 정보, 잡 모니터링, Billing 상태를 관리하는 화면.
REQUIRED

Theme App Extension

📍 판매자 스토어프론트 (고객)
Liquid app block. 제품 페이지에 "🎨 디자인하기" 버튼 추가. 클릭 시 iframe으로 Storige 편집기 진입.
RECOMMENDED

App Proxy

📍 판매자 도메인 (Shopify HMAC)
shop.com/apps/storige/orders/{id}/pdf 형태. 결과 PDF 다운로드를 판매자 도메인으로 프록시 (Shopify 인증 후).

5 단계별 로드맵 (총 6~8주)

사전 등록 + 환경 1일

  • Shopify Partner 계정 가입 ($19 1회)
  • Development store 생성 (무료)
  • App 등록 → API key + secret 발급
  • 개발 도메인 결정 (shopify.papascompany.co.kr)

임베디드 앱 신규 구축 1주

  • apps/shopify 워크스페이스 (Remix + shopify-app-remix)
  • OAuth 흐름 (install → consent → token exchange)
  • OAuth 콜백에서 Site 자동 등록 (Phase A 활용)
  • Polaris UI: 잡 목록 / 사이트 설정 / Billing

백엔드 통합 3일

  • apps/api에 /shopify/oauth/callback + webhook 라우트
  • HMAC-SHA256 검증 어댑터 (Base64 sig와 분리)
  • GDPR webhooks 3종: customers/data_request / customers/redact / shop/redact
  • orders/paid → 워커 자동 합성 잡 발사
  • App proxy endpoint: /apps/storige/orders/{id}/pdf

Theme App Extension 3일

  • Liquid app block storige-designer.liquid
  • 제품 페이지에 "🎨 디자인하기" 버튼 + iframe modal
  • 편집기 완료 → postMessage → cart line item property 첨부

Billing + 가격 정책 2일 v1.1 결정

  • Shopify Billing API: RecurringApplicationCharge + UsageCharge (capped)
  • Free tier: 합성 잡 20건/월 무료
  • 사용량: 21건째부터 $0.50/건 (월 $99 cap)
  • Pro: $29/월 무제한 (선택)
  • Polaris UI에 실시간 사용량 게이지 + 80% 임계 알림

Built for Shopify 인증 준비 1주

  • Lighthouse ≤10pt 영향 검증 (별도 도메인 통과 쉬움)
  • GDPR/CCPA 정책 페이지 + 개인정보 처리방침
  • 접근성 (WCAG 2.1 AA) — Polaris 자동
  • App listing 자료: 스크린샷, 데모 영상(영문 자막), 아이콘

검수 제출 + 1차 review 대응 2~4주

  • Shopify Partner Dashboard에서 submission
  • 평균 review 4-7 영업일 → feedback → 수정 → 재제출
  • 평균 2-4주 후 publish
