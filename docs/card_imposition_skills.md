# 카드/명함 특화 양면 조판 기술 규격 (Card Duplex Imposition Skills)

이 문서는 각기 다른 디자인의 낱장 카드를 디지털 인쇄용 원단에 배치하고, 관리자가 설정한 좌표에 명함재단기 센서 마커를 포함하여 양면 조판하는 워커의 핵심 로직을 정의한다.

## 1. 조판 환경 및 설정 (Configuration)

### 1.1 관리자 정의 마커 (Configurable Marker)
센서 마커는 재단기 모델에 따라 위치와 크기가 다르므로, 워커는 다음 설정값을 참조하여 동적으로 생성한다.
- `use_sensor_marker`: 마커 생성 활성화 여부
- `marker_x / marker_y`: 원단 왼쪽 에지 기준 마커 시작 좌표 (mm)
- `marker_width / marker_height`: 마커의 가로/세로 크기 (mm)
- `marker_color`: 마커 색상 (기본값 K100)

### 1.2 원단 및 판형
- 워커는 사전에 등록된 원단 규격(A3, SRA3 등)과 해당 원단에 배치될 카드의 수량 정보를 인지해야 한다.

## 2. 양면 조판 및 마커 생성 규칙

### 2.1 센서 마커 생성
- 워커는 조판 완료 후, 최상위 레이어에 관리자가 지정한 `(x, y)` 좌표와 `(width, height)` 크기로 검정색 직사각형 마커를 드로잉한다.

### 2.2 양면 조판 알고리즘
- **앞면 (Front Side)**: 순차 배치 및 관리자 설정 위치에 마커 생성.
- **뒷면 (Back Side)**: 앞면 레이아웃을 가로축 기준으로 **좌우 반전(Horizontal Mirroring)**.
  - 앞면의 `j`번째 열은 뒷면에서 `(Total_Columns - j + 1)`번째 열로 이동한다.
  - 뒷면 마커 생성 여부는 재단기 사양에 따라 결정하되, 기본적으로 앞면 마커를 기준으로 정합을 맞춘다.

## 3. 워커 처리 체크리스트
1. [ ] **설정값 로드**: 관리자가 지정한 마커 좌표와 크기 값을 정상적으로 읽어왔는가?
2. [ ] **동적 마커 드로잉**: 하드코딩된 좌표가 아닌 설정된 값에 마커를 그렸는가?
3. [ ] **양면 반전**: 뒷면 조판 시 앞면의 열(Column) 좌표가 정확히 반전되었는가?
