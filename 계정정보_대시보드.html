  • Storige 백엔드 자동 배포 없음 — API/Worker 수정 시 docker compose up -d --build 수동 실행 필요. 프론트만 자동 배포됨.
  • Bookmoa-Mobile 도메인 미등록 — vercel.app 무료 도메인 사용 중. 운영 전환 시 커스텀 도메인 + 결제 연동 + 알림 인프라 별도 필요.
  • 시크릿 보관 — .env, SSH 키, Supabase service key 등은 절대 커밋 금지. CLAUDE.local.md 는 .gitignore 확인.
